)Назначение: база данных частных объявлений о продаже или дарении объектов

Категории покупателей: физические и юридические лица
Категории продавцов: физические и юридические лица

Объект: товар или услуга

Ограничения:
1) Время показа объявления ограничено 30 календарными сутками;
2) Условия передачи объекта: возмездное, безвозмездное;
3) Для возмездной передачи объекта цена фиксированая;
4) Валюта - рубли;
5) Объект может содержать до 5 изображений суммарным объемом 5 Мб или одно видео того же объема 
6) Я не стал включать сущности, связанные с бухгалтерией, на это потребовалось бы еще столько же таблиц

Сущности:
1) + Пользователь
2) + Взамодействие с пользователем - пользователь может иметь один и более способов взаимодействия;
3) + Справочник типов взаимодействий с пользователем - необходим для визуализации способа взаимодействия в интерфейсе, а так же программной реализации данного взаимодействия;
4) + Профиль юридического лица - содержит атрибуты, характерные для юридического лица;
5) + Профиль физического лица - информация о физ. лице;
6) + Отзывы о продавце - возможность оставить рейтинг для пользователя (насколько он надежен);
7) + Категории товаров - иерархический справочник категорий товаров услуг; 
8) + Набор характеристик объекта - поименованный набор характеристик для заданной категории товара, позволяет проводить сравнение объектов по одинаковым характеристикам. Характеристики набираются вверх ко категориям
9) + Единица измерения характеристики объекта - справочник единиц измерения характеристики объекта
10) + Объект
11) + Файловок хранилище - содержит изображения объекта, аватарки (фирменные знаки для ЮрЛ), а так же вспомогатеотные иконки
12) + Допустимые типы файлов - содержит справочнник MIME-типов файлов
13) + Значения характеричстик объекта
14) + Сообщения к объекту (вопрос/ответ)
15) + Справочник стран
16) + Справочник регионов (территориальное деление страны)
17) + Справочник населенных пунктов - важен, поскольку основной способ передачи объектов - самовывоз
